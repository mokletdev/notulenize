# notulenize

A Mobile App for summarizing your meeting audio

## How to run

```bash
git clone https://github.com/mokletdev/notulenize.git
```

Install dependencies

```bash
flutter pub get
```

Initialize API key

`.env`
```bash
API_KEY = "Your API key"
```

Run the app

```bash
flutter run
```
